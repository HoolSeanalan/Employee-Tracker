## Description
An command line program used to view a saved employee database as well as add new departments, roles, and employees and update employee roles.
## Installation
'npm i', 'source db/schema.sql', (optional) 'source db/seeds.sql'
## Usage
Navigate the menus using arrow keys and enter.
